- **Encryption:** AES-256 (CBC)
- **Keys:** PBKDF2 (100,000 iterations, salt)
- **Integrity:** HMAC-SHA256
- **Each file:** unique salt and IV
- **Passwords:** never stored

3. Select file or folder
4. Enter password (remember it!)
